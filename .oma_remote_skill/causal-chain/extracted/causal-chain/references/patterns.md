# 常见金融传导链模式库

模型生成传导链时，先查阅此文件找到匹配的骨架模式，再根据用户具体场景调整。

## 1. 货币政策 — 加息链

```
央行加息 →|利率传导| 无风险利率上升
央行加息 →|汇率机制| 本币走强
无风险利率上升 →|DCF传导| 成长股估值压缩
无风险利率上升 →|融资成本| 信贷条件收紧 → 房地产承压
本币走强 →|资本流动| 新兴市场资金外流 → 新兴市场股汇承压
本币走强 →|计价效应| 大宗商品承压 → 资源股利润收窄
```

**多空方向：**
- 利空：成长股、房地产、新兴市场、大宗商品、高杠杆企业
- 利好：银行股（息差扩大）、本币资产、短期国债

## 2. 货币政策 — 降息链

```
央行降息 →|利率传导| 无风险利率下降
央行降息 →|汇率机制| 本币走弱
无风险利率下降 →|DCF传导| 成长股估值扩张
无风险利率下降 →|融资成本| 信贷宽松 → 房地产复苏
无风险利率下降 →|资金迁移| 存款搬家 → 股市流入
本币走弱 →|出口竞争力| 出口企业受益
```

**多空方向：**
- 利好：成长股、房地产、出口企业、黄金
- 利空：银行股（息差收窄）、进口依赖型企业

## 3. 贸易摩擦链

```
加征关税 →|成本传导| 进口成本上升 → 下游涨价 → 消费需求萎缩
加征关税 →|倒逼效应| 国产替代加速 → 本土龙头受益
加征关税 →|供应链重构| 产能转移至第三国
加征关税 →|避险情绪| 市场风险偏好下降 → 避险资产受益
```

**多空方向：**
- 利空：依赖进口的制造商、消费电子、全球化供应链企业
- 利好：国产替代龙头、避险资产（黄金/国债）、第三国制造商

## 4. 供给冲击链（如油价暴涨）

```
供给中断 →|供需缺口| 商品价格飙升
商品价格飙升 →|成本推动| 生产成本上升 → PPI上涨 → CPI传导
商品价格飙升 →|利润挤压| 下游企业利润承压
CPI传导 →|政策约束| 央行被迫收紧 → 滞胀风险
```

**多空方向：**
- 利好：上游资源股、替代能源
- 利空：下游制造业、航空运输、消费品

## 5. 信用事件链（如大型违约）

```
企业违约 →|信用利差| 同行业债券抛售
企业违约 →|风险传染| 交易对手风险暴露 → 银行坏账预期上升
信用利差走阔 →|融资断裂| 弱资质企业再融资困难 → 连锁违约
银行坏账预期 →|估值压力| 银行股下跌
连锁违约风险 →|政策博弈| 央行是否救市 → 流动性注入预期
```

## 6. 科技/产业政策链

```
产业政策出台 →|补贴驱动| 目标行业产能扩张
产业政策出台 →|预期驱动| 相关股票估值提升
产能扩张 →|供给过剩| 行业价格战 → 弱势企业出清
产能扩张 →|上游拉动| 设备/材料需求增长
```

## 7. 地缘政治链

```
地缘冲突升级 →|避险情绪| 黄金+美债上涨
地缘冲突升级 →|供给担忧| 能源/粮食价格上涨
地缘冲突升级 →|风险偏好| 股市下跌（尤其是冲突相关地区）
能源价格上涨 →|通胀传导| → 参考"供给冲击链"
```
