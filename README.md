## 🚦 Getting Started

### Prerequisites
- Node.js 18+
- Supabase Account
- Stripe Account

### Environment Configuration
Rename `.env.example` to `.env.local` and configure your node parameters:
```bash
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
STRIPE_SECRET_KEY=your_key
AI_ORCHESTRATOR_KEY=your_key
```

### Installation
```bash
npm install
npm run dev
```

### Build Sequence
```bash
npm run build
```
